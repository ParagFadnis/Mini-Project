Animal Welfare Management System
The Animal Welfare Management System is a web-based platform developed to help citizens report injured or stray animals, allow volunteers to respond to rescue cases, and enable administrators to monitor and manage animal welfare activities. The system provides role-based login and registration for Admin, Public Users, and Volunteers. It also supports report tracking, adoption management, and email notifications for newly submitted reports.
1. Technology Stack and Tools Used
•	Frontend: HTML5, CSS3, JavaScript
•	Backend: Node.js, Express.js
•	Database: MongoDB with Mongoose
•	Authentication: JWT (JSON Web Token)
•	Email Notifications: Nodemailer
•	Version Control: Git and GitHub
•	Development Tools: VS Code, Live Server, Postman
2. Features and Functionalities Implemented
•	Role-based Login System for Admin, Public Users, and Volunteers
•	User Registration for Public Users and Volunteers
•	JWT-based Authentication and Authorization
•	Animal Reporting System with image upload, description, and user location
•	Automatic Email Notification to Admin when a report is submitted
•	Report Tracking System for users
•	Admin Dashboard to view and update report status
•	Volunteer Dashboard to accept and complete rescue reports
•	Animal Adoption Management
•	Animal Safety Information Section
3. Installation and Execution Steps
1.	Clone the GitHub repository using: git clone <repository-link>
2.	Open the project folder in Visual Studio Code
3.	Install all required dependencies using: npm install
4.	Start MongoDB service on your system
5.	Run the backend server using: node server.js
6.	Open frontend files using Live Server or localhost
7.	Register a new user account and login based on role
8.	Use the dashboards to access project functionalities
